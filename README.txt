"Old Pages Review Service" plugin represents Confluence scheduled job with GUI. The plugin checks modification dates of
pages and sends mail to creators. You can define Space keys that should be monitored, groups that contain creators and
period to consider pages as outdated in days. Additionally, it is possible to set your own custom mail subject and body.
Available variables for mail body: $creator, $days, $links. Visit Administration - Scheduled Jobs in order to set a
time interval for a running job or to run it manually.
